## ◆ Featured Project — HeroSync

<div align="center">

[![HeroSync](https://github-readme-stats.vercel.app/api/pin/?username=Yug-Dave020&repo=HeroSync&theme=tokyonight&border_color=00d9ff&icon_color=00d9ff&title_color=00d9ff)](https://github.com/Yug-Dave020/HeroSync)

</div>

> **HeroSync** — A production-grade gamified productivity suite with a full AI companion system.

| Layer | Technology |
|-------|-----------|
| **Backend** | Spring Boot 3.5 · Spring Security · JPA · GraphQL + REST hybrid |
| **Frontend** | Vue 3 (Composition API) · Pinia |
| **AI Engine** | Multi-provider: Groq (LLaMA 3.1) · Gemini Flash · GPT-4o · swappable at runtime |
| **Infrastructure** | Docker · Nginx · MySQL · Oracle Cloud VPS · Health-checked containers |

**The AI Companion System**

Three AI characters with distinct personalities — all reading live user data:

```
SYNC  ·  The Strategist    ·  "Your streak is 8 days. 180 XP to Level 2. Accelerate."
AURA  ·  Empathetic Guide  ·  "You showed up 6 out of 7 days — that's real dedication."
VOLT  ·  The Motivator     ·  "8 DAY STREAK. BLAZING HOT. Don't let it cool down TODAY."
```

**How it works technically:**

- Single `AiChatService` orchestrates context building → prompt selection → provider routing
- `AiContextBuilder` injects live stats (streak, XP, quests, missions, heatmap) into every prompt
- `AiPromptBuilder` switches personality system prompts per companion x mode (chat / onboarding / coaching / diagnostics)
- Provider interface (Groq → Gemini → GPT) is swappable per user from Settings — zero code change
- Performance Diagnostics: AI returns structured JSON → rendered as S-Rank strength cards in real time
